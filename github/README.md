# Batch Markdown Converter repository README files

각 언어 저장소의 루트에 다음 파일을 `README.md`라는 이름으로 복사합니다. 현재 프로젝트
루트 `README.md`는 영문 공개 저장소용이며 기존 한국어 개발 문서는 `README.ko.md`에 있습니다.

- `Korean/README.md`: 한국어판 GitHub 저장소용
- `English/README.md`: 영문판 GitHub 저장소용

두 README는 동일한 기능과 배포 정책을 설명하며 실행 파일명, DMG 이름, 실행 스크립트만
각 언어판에 맞게 구분되어 있습니다.
