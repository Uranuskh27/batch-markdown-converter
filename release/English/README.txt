Batch Markdown Converter English (Apple Silicon)

Installation
1. Open the DMG.
2. Drag Batch Markdown Converter English.app into the Applications folder.
3. Launch Batch Markdown Converter English from Applications.

macOS may block a test build that has not been notarized by Apple. Verify the file against
SHA256SUMS.txt and make sure it came from a trusted official release before Control-clicking the
app in Finder and choosing Open.

Requirements: macOS 13 or later, Apple Silicon (arm64)
