Batch Markdown Converter 한국어판 (Apple Silicon)

설치 방법
1. DMG를 엽니다.
2. Batch Markdown Converter Korean.app을 Applications 폴더로 드래그합니다.
3. 응용 프로그램 폴더에서 Batch Markdown Converter Korean을 실행합니다.

이 빌드가 Apple 공증되지 않은 테스트 빌드라면 macOS가 실행을 차단할 수 있습니다.
신뢰할 수 있는 공식 배포처에서 받은 파일인지 SHA256SUMS.txt로 먼저 확인한 뒤,
Finder에서 앱을 Control-클릭하고 '열기'를 선택할 수 있습니다.

지원 환경: macOS 13 이상, Apple Silicon(arm64)

오픈소스 안내
이 앱은 Microsoft의 공식 제품이 아닙니다. MarkItDown은 Microsoft의 MIT 라이선스로,
Qt/PySide는 LGPLv3 조건으로 사용합니다. 전체 라이선스, 대응 소스 안내 및 라이브러리
교체 방법은 DMG의 Third-Party-Licenses, SOURCE_OFFER.md, BUILD_AND_RELINK.md를 확인하세요.
