# LGPL corresponding source

This binary distribution uses unmodified Qt for Python, PySide6, Shiboken6,
Qt Base, and Qt SVG version 6.11.1 under LGPL version 3.

The exact corresponding source archives are distributed in the `Source-Code`
folder of the same GitHub Release as the DMG:

- `pyside-setup-everywhere-src-6.11.1.tar.xz`
- `qtbase-everywhere-src-6.11.1.tar.xz`
- `qtsvg-everywhere-src-6.11.1.tar.xz`

Their SHA-256 values are provided in `LGPL-Sources-SHA256SUMS.txt`. The release
maintainer will keep these source archives available for at least three years
after the corresponding binary release.

If a binary copy was provided without the source archives, open an Issue in
the official GitHub repository for this application and request the source for
the exact release version. It will be supplied at no more than the reasonable
cost of physically providing it.

See `BUILD_AND_RELINK.md` for installation information and library replacement
instructions. No license term of this application restricts reverse
engineering performed for debugging modifications to LGPL-covered libraries.
